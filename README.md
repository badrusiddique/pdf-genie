# pdf-genie

> Every PDF tool, beautifully simple.

An open-source PDF toolkit with 27 tools — merge, split, compress, convert, edit, sign, and more. Built with Next.js 14, TypeScript, and pdf-lib.

## Stack

- **Frontend:** Next.js 14 App Router + TypeScript
- **Styling:** Tailwind CSS + Framer Motion
- **PDF Processing:** pdf-lib (client), pdfjs-dist (render), Tesseract.js (OCR)
- **Testing:** Vitest + Playwright
- **Deployment:** Vercel

## Getting Started

```bash
pnpm install
pnpm dev
```

## Environment Variables

Copy `.env.local.example` to `.env.local` and fill in values.
